# Aegis AI Fleet

AI-powered fleet safety intelligence platform for vehicle exception analysis, risk assessment, and actionable recommendations.

Developed as part of an internship project at **Kriti Labs**.

---

#  Overview

Aegis AI Fleet is a fleet monitoring and analytics platform designed to help organizations analyze vehicle exceptions, identify safety risks, and generate intelligent recommendations for improving fleet operations.

The platform processes fleet exception datasets and provides interactive insights through a dashboard and AI-assisted analysis.

---

#  Features

* 📊 Fleet exception analytics
* 🚗 Vehicle-wise exception monitoring
* ⚠️ Risk and safety assessment
* 📈 Exception trend visualization
* 🤖 AI-powered recommendations
* 📋 Structured tabular responses
* 🔍 Exception breakdown analysis
* 📉 Fleet-wide insights and reporting

# Getting Started

## Clone Repository

```bash
git clone https://github.com/Harsheys27/Aegis-AI-Fleet.git
cd Aegis-AI-Fleet
```

## Backend Setup

```bash
cd chatbot-backend

pip install -r requirements.txt

python app.py
```

Server runs on:

```text
http://localhost:5000
```

## Frontend Setup

Open:

```text
chatbot-app/index.html
```

or run using VS Code Live Server.
